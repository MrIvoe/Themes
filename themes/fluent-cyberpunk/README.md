# Fluent Cyberpunk Theme

`fluent-cyberpunk` is the flagship dark theme that combines Fluent desktop structure with cyberpunk accent strategy.

## Direction

- Fluent foundations: clear shell hierarchy, composable panels, practical desktop spacing.
- Cyberpunk accents: electric cyan primary, toxic yellow secondary, magenta reserved for high-energy and critical states.
- Interactive glow is limited to focus, selected, and active controls.

## Visual DNA

- Mood: tactical, futuristic, electric
- Shape language: sharp-tech
- Surface style: premium dark glass
- Accent strategy: electric cyan primary + toxic yellow secondary + magenta special
- Motion: quick, precise interactions

## Accessibility

- Visible focus ring required for interactive controls
- Contrast target: WCAG AA baseline with additional control-state checks in validator
- Reduced motion preset: `minimal`

## Files

- `theme.json`: foundation tokens and scales
- `semantic.json`: semantic role mapping
- `components.json`: singular component contracts and variants
- `icons.json`: icon pack mapping and role color wiring
- `resources.json`: runtime family selection, support levels, and family DNA
- `preview-manifest.json`: reusable preview fixtures to render for visual QA
